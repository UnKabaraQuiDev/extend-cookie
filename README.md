# Extend Cookie

Extends the duration of cookies set by domain.

## How to use
1. Install the extension
2. Click on it in the "Extensions" dropdown in the top bar
3. Add you domains, wildcards `*` are supported

Examples:
* `*.education.lu 7D 0H 0M 0S`
* `*.microsoft.com 7D 0H 0M 0S`

### Firefox extension
Firefox store: https://addons.mozilla.org/en-US/firefox/addon/extend-cookies/
